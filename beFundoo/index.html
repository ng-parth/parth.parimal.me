Fundoo Stock Application

1) This app shows stock prices.
2) If user is logged in, he can add or update existing stocks.
3) If user tries to edit stocks that are not in the list, he will be warned and redirected to stock list.
4) While edit, user cannot updates id to any existing id.
5) User cant add new stock with id of any existing record.
6) If user tries to access create or edit page without Signing In, he will be redirected to list page with a warning.
7) If user is already logged in, he cant visit LOGIN page and will be redirected back to LIST page.
8) User cannot submit any form unless all required fields are filled in forms CORRECTLY.
9) User credentials are case sensitive.

NOTICE:
1) User login will persist even after closing browser.
2) You need to explicitly click 'SIGN OUT' to logout any user.
